## 專案簡介	
此專案是一個可讓使用者製作、管理多份履歷的 Web 應用系統。
使用者可以建立履歷、儲存草稿、編輯內容、投遞不同公司，並於右側卡片界面管理已建立的多份履歷。
系統同時提供即時預覽、深色模式、表單驗證、多筆工作經驗新增、證件照上傳與多欄位履歷內容輸入等功能。

## 使用技術	
- HTML
- CSS3 / Bootstrap 5
- JavaScript
- DOM 操作
- 事件委派
- Constraint Validation API
- setCustomValidity()
- localStorage 
- Front-end 功能模組化
  
## 功能特色
### 1. 完整履歷編輯表單
- 包含：
  - 姓名、Email、電話  
  - 學歷（含學校欄位）  
  - 目標行業 / 期望職位 / 應徵公司  
  - 技能標籤（事件委派）  
  - 證件照上傳與預覽  
  - 自我簡介、自傳  
  - 語言能力  
  - 證照  
  - 多筆工作經驗  
### 2. 即時預覽
- 輸入左側欄位時，右側同步呈現：
  - 個人資訊  
  - 學歷  
  - 行業 / 職位 / 公司  
  - 技能標籤  
  - 語言能力  
  - 工作經驗卡片  
  - 證件照  
  - 自傳、證照內容  
- 深色模式全區支援亮白字  
### 3. 表單驗證
- blur 事件啟動錯誤提示  
- input 時即時更新與清除錯誤  
- `setCustomValidity()` 產生自訂中文錯誤訊息  
- 送出時阻擋第一個錯誤欄位  
### 4. 履歷儲存
- 可儲存多份履歷  
- 支援草稿模式  
- 支援編輯既有履歷  
- 支援刪除履歷  
### 5. 已建立履歷卡片列表
- 以卡片式顯示每份履歷：
  - 第 N 份履歷  
  - 草稿 / 已投遞 標籤  
  - 姓名、聯絡資訊、技能  
  - 編輯 / 刪除按鈕  
### 6. 深色模式
- 背景、卡片、表單、預覽區全套支援  
- 即時預覽文字全亮色防止看不見  
- 自動記錄使用者模式（localStorage）  
### 7. 送出防重複
- 送出時鎖定按鈕並顯示 Loading  
- 模擬投遞成功後顯示提示訊息
